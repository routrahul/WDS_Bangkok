Canvas Based Visualizations

Canvas Based Visualizations

The developer works out the commands to draw objects, create the model or scene

Takes care of the updating

The browser’s Graphics API communicates these drawing commands, browser simply executes them

You draw your pixels and the system forgets all about them thus, cutting down on the extra memory needed to maintain an internal model

WebGL based Visualizations

WebGL based Visualizations

Enables web content to use an API based on OpenGL to perform 2D and 3D rendering in an HTML canvas

Programs consist of control code written in Javascript and shader code which is executed on a computer’s GPU

WebGL elements can be mixed with other HTML elements and can be composited with other parts of the page or backgrounds
